# 学習資料 - 前処理 - 

## 概要
以下の学習補助資料を参考にして学習を進めてください。<br>
前処理とは、生データからその特徴を数値で表した「特徴量(入力に使用するデータ)」にすることです。
機械学習の観点ではテキストデータは役に立たず、意味のある数値に変換できて初めて機械学習アルゴリズムに入力することができます。
